## 项目简介

0-Anchor 是一款基于 Android 无障碍服务的自律辅助工具。旨在帮助用户在工作/娱乐场景切换时保持清醒认知，通过「任务模式」与「娱乐模式」的强制选择机制，辅助用户改善手机使用习惯。

## 名字由来

| 字符 | 含义 |
|------|------|
| **0** | 排序靠前，系统设置列表中一目了然 |
| **Anchor** | 锚，寓意在信息洪流中稳住自己 |

## 核心功能

### 智能拦截
- 自动检测目标应用的打开行为
- 弹出模式选择窗口，强制用户做出选择

### 任务模式 ⏱️
- 输入任务名称，开始正向计时
- 气泡实时显示计时与任务名称
- 点击气泡随时退出

### 娱乐模式 🎮
- 快速选择时长（5 / 10 / 15 / 30 / 60 分钟）
- 倒计时结束自动暂停
- 可配置冷却时间（60-180 分钟），冷却期内娱乐按钮灰显

### 气泡拖动
- 计时气泡支持拖动至屏幕任意位置

## 权限说明

| 权限 | 用途 |
|------|------|
| 无障碍服务 | 检测应用切换，触发拦截窗口 |
| 悬浮窗 (SYSTEM_ALERT_WINDOW) | 展示拦截界面与计时气泡 |
| 通知 (POST_NOTIFICATIONS) | 常驻通知保活，防止后台被清理 |
| 查询已安装应用 (QUERY_ALL_PACKAGES) | 选择需要管理的应用 |

> ⚠️ 建议同时开启电池优化与自启动权限，确保后台稳定运行。
