# 贡献指南

感谢您对 0-Anchor 的关注！

## 如何贡献

1. **Fork 本项目**
2. **创建特性分支** (`git checkout -b feature/amazing-feature`)
3. **提交更改** (`git commit -m 'Add some amazing feature'`)
4. **推送到分支** (`git push origin feature/amazing-feature`)
5. **创建 Pull Request**

## 提交信息规范

- 使用清晰简洁的描述
- 开头使用动词（Add/Fix/Update/Remove）
- 示例：`Add cooldown time configuration`

## 问题反馈

请通过 Issues 功能反馈问题，包含以下信息：
- Android 版本
- 手机型号
- 问题描述
- 复现步骤

## 代码规范

- 遵循 Android Kotlin/Java 编码规范
- 添加必要的注释
- 确保代码可读性

## 许可证

提交代码即表示您同意您的贡献将遵循本项目的 MIT 许可证。
